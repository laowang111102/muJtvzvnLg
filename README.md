### 主要功能实现

1. **登录注册功能**：用户需完成注册和登录才能使用系统的其他功能。
2. **首页展示功能**：用户无论是否登录，均可访问网站首页。
3. **个人信息展示功能**：用户可查看自己的基本信息和权限等。
4. **用户信息管理功能**：管理员用户可对注册用户的信息进行维护。
5. **修改密码功能**：用户可在前台页面修改自己的密码。
6. **在线聊天功能**：用户与系统进行智能对话。
7. **问答列表功能**：查询所有聊天和问答记录。
8. **改变主题功能**：切换不同的界面主题和字体。
